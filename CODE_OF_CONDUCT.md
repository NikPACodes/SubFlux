# Правила поведения

Мы стремимся к дружелюбной и уважительной атмосфере в проекте SubFlux.

## Основные принципы
- Уважительное общение
- Отсутствие токсичности, оскорблений и дискриминации
- Конструктивная критика вместо личных нападок

## Недопустимое поведение
- Агрессия, троллинг, хамство
- Личные оскорбления
- Публикация личной информации без согласия

## Меры
Владелец проекта оставляет за собой право:
- Удалять неприемлемый контент
- Ограничивать участие пользователей, нарушающих правила

Этот документ создан для защиты участников и здорового развития проекта.
